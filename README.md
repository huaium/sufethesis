# SUFEThesis

**上海财经大学本科毕业论文 $\LaTeX$ 模板**

- 请使用 XeLaTeX 编译
- 基于模板 [sufethesis](https://github.com/Lujia-Bai/sufethesis) 进行修改
- 修正在某些平台上的下划线问题
- 必要时尝试替换字体再编译
